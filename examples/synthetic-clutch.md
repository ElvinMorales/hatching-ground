# Synthetic clutch: Bounded household helpers

> Synthetic public-safe example. Do not treat this as real personal data, memory, state, or implementation output.

All candidates serve fictional users and use invented input only.

## Garden Note Formatter

- **One-line idea:** Turn one unstructured fictional garden note into a dated Markdown checklist for review.
- **Why useful:** A fictional home gardener can see the next steps without rereading scattered prose.
- **First usable product:** Paste or type one synthetic note, review extracted tasks, and export a dated checklist.
- **Likely complexity:** Low; a bounded local/file-first workflow with one input and one output.
- **Obvious non-goals:** Weather monitoring, purchasing, reminders, device control, plant disease diagnosis, provider calls, backend services, databases, and broad filesystem access.

## Recipe Tag Cleaner

- **One-line idea:** Normalize tags on a small set of fictional recipe cards.
- **Why useful:** Consistent tags make synthetic recipe examples easier to scan.
- **First usable product:** Paste a fictional recipe and export a reviewed Markdown card with tags from a fixed vocabulary.
- **Likely complexity:** Low to medium because the vocabulary needs a separate decision.
- **Obvious non-goals:** Meal planning, nutrition or health advice, grocery ordering, scraping, account access, and recipe storage.

## Household Checklist Organizer

- **One-line idea:** Group fictional household tasks into a simple reviewable checklist.
- **Why useful:** It can reduce repetition in a synthetic task list.
- **First usable product:** Paste invented tasks and export them under fixed categories.
- **Likely complexity:** Medium because category and recurrence rules are not yet bounded.
- **Obvious non-goals:** Real household routines, family assignments, scheduling, reminders, messaging, device control, and persistent profiles.

## Selection

**Garden Note Formatter** is selected because its single-user, single-input, single-output loop is clearest. It demonstrates artifact-first incubation without sensitive data or operational integration. Recipe Tag Cleaner needs a tag-vocabulary decision, while Household Checklist Organizer should be narrowed before hatching.
